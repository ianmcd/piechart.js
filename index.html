piechart.js

piechart.js is an extremely lightweight & simplistic JQuery script for building css3 pie charts on the fly. This script originally spawned from a script that I created when attempting returning search result relevancy in a creative way.

How to use

demonstration of piechart.js in action

piechart.js is amazingly simple to use!

Step 1: link jquery to relevant pages: <script src="http://code.jquery.com/jquery-latest.min.js"></script>

Step 2: upload and link piechart.css & piechart.js to relevant pages: <script src="js/piechart.js"></script>
<link rel="stylesheet" type="text/css" href="css/piechart.css" />

Step 3: add the following bit of code to the page to create your first pie chart(seen above): <div class="pie" title="8% 11% 6% 35% 25% 15%"></div>

Step 4: edit piechart.css to style as you see fit.

Additional Info

piechart.js allows you to input as many variables as you like given that they do not surpass a sum of 100%. Going beyond 100% will return an error.

The .css file is set up for a maximum of 6 variables, if you plan on using more than 6 you will need to add the following code to the css file with a background color of your choosing: div.pie div.fill7 { background-color: #8fc5f0; } fill# corresponds to the variable number on the chart(in this case, 7). If you wish to have 10+ variables you would need to fill in the blanks for fill7, fill8, fill9, fill10 and so forth. This is indicated by the comments in piechart.css

If at any point you change the class names in piechart.css you will have to do the same in piechart.js to ensure things work correctly. This will probably change in future versions.
